MVC Controllers: ``rafcon.gui.controllers``
===========================================

All controllers of the MVC architecture.

The controllers are the interface between the models (holding the data) and the view (showing the data). They are
responsible for the logic.

All controllers of RAFCON inherit from :class:`rafcon.gui.controllers.extended_controller.ExtendedController`, which
inherits from the controller class of GTKMVC.
